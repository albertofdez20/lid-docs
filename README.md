# Welcome to LID

**Your customers are your sales team.**

Create a product. Share a link. Get paid instantly.

LID is a programmable commerce routing layer built on Solana. Anyone can create a digital product, anyone can share the link, and when it sells, the seller, the affiliate, and LID get paid in the same transaction.

No approvals. No contracts. No 30 to 60 day payment delays. Flat 3% infrastructure fee.

## Who this site is for

These docs are for four groups.

**Sellers** creating digital products and inviting others to distribute them. Start with [Getting Started](getting-started/what-is-lid.md), then read [For Sellers](for-sellers/create-a-product.md).

**Affiliates** sharing links and earning on every sale. Go to [For Affiliates](for-affiliates/how-affiliate-works.md).

**Developers and agent builders** who want to integrate LID as economic infrastructure for autonomous software. Start at [Developers Overview](developers/overview.md).

**Investors, partners, and anyone curious about where this is going.** Read the [Vision](company/vision.md) and [Roadmap](company/roadmap.md).

## The core loop

Everything LID does serves one loop.

| Step | What happens |
|---|---|
| **Create** | A seller sets up a digital product in minutes. |
| **Share** | A link is distributed with on-chain attribution. No manual affiliate contracts. |
| **Sell** | A real transaction closes. Attribution binds at checkout. |
| **Get Paid** | Seller, affiliate, and LID settle in USDC in the same transaction. |

If a feature strengthens this loop, we build it. If it doesn't, we don't.
