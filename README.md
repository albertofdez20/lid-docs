# Welcome to Lid

**Your audience is your sales team.**

Lid is a routing layer for commerce. Create a product. Share a link. When the sale clears, the seller, the affiliate, and Lid settle in the same transaction. 3% flat fee. Digital dollars. On Solana.

No approvals. No contracts. No 30-day payout delays.

## The core loop

Everything Lid does serves one loop. If a feature strengthens it, we build it. If not, we don't.

| Step         | What happens                                                       |
| ------------ | ------------------------------------------------------------------ |
| **Create**   | A seller sets up a digital product in minutes.                     |
| **Share**    | Any visitor generates an affiliate link. Attribution is on-chain.  |
| **Sell**     | Checkout runs. Attribution binds at the moment of purchase.        |
| **Get paid** | Seller, affiliate, and Lid settle in digital dollars in the same transaction. |
